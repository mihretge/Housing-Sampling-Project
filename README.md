# Analysis of Housing Prices in 45 Major Canadian Cities Using Sampling Methods

## Overview
This project analyzes housing prices across 45 major Canadian cities using sampling methods from survey sampling theory. The goal is to estimate the population mean and total housing prices and compare the performance of different sampling methods.

## Files
- `housing prices canada.csv` — dataset used in the analysis
- `housing sampling analysis.Rmd` — R Markdown file with the code
- `final report.pdf` — final written report

## Main finding
Stratified sampling, especially Neyman allocation, gave the best results, while cluster sampling performed the worst for this dataset.
